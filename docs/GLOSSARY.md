# GLOSSARY.md — 프로젝트 용어 사전

> IQ Spark Studio 프로젝트에서 사용하는 전용 용어, 약어, 개념을 정의합니다.

## A

| 용어 | 정의 |
|------|------|
| **A1** | Answer 1. 첫 번째 문제(P1)의 정답 해설 프레임 |
| **AI 라벨** | TikTok에서 AI 생성 콘텐츠임을 표시하는 내장 라벨 (RULE 8) |
| **Amber Gold** | 브랜드 강조 색상 `#F59E0B` |
| **analysis.json** | 퍼즐 분석 결과를 저장하는 JSON 파일 |

## B

| 용어 | 정의 |
|------|------|
| **BGM** | Background Music. 배경음악. 대표님 제공 시에만 적용 (RULE 17) |
| **브랜드 템플릿** | 딥블루 배경 + 골드/시안 테두리 + Inter 폰트의 표준 프레임 디자인 |

## C

| 용어 | 정의 |
|------|------|
| **Comment Mode** | Q2(Hard 문제)에서 정답을 영상에서 공개하지 않고 댓글로 유도하는 전략 |
| **copy_5platform.json** | 5채널 맞춤 카피(제목/캡션/해시태그)를 저장하는 JSON 파일 |
| **CTA** | Call to Action. 시청자에게 특정 행동을 유도하는 문구 (댓글, 사이트 방문 등) |
| **Claude Sonnet 4.6 (thinking)** | 카피라이팅 엔진으로 사용하는 Anthropic의 LLM. thinking 모드로 심층 추론 기반 고품질 카피 생성 |

## D

| 용어 | 정의 |
|------|------|
| **Dark Navy** | 브랜드 주 배경 색상 `#0F0F1A` |
| **daily_summary.md** | 게시 48시간 후 성과 데이터를 정리한 보고서 |

## E

| 용어 | 정의 |
|------|------|
| **Easy (P1)** | 첫 번째 문제. 상대적으로 쉬운 난이도. Reveal Mode로 정답 공개 |
| **8-Phase** | 소재분석 → 트렌드리서치 → 카피라이팅 → 프레임생성 → 영상렌더링 → 플랫폼최적화 → 업로드 → 성과보고 |
| **에스컬레이션** | 에이전트가 자동 처리 불가한 상황 발생 시 대표님에게 보고하는 절차 |

## F

| 용어 | 정의 |
|------|------|
| **FFmpeg** | 영상 렌더링에 사용하는 오픈소스 멀티미디어 프레임워크 |
| **Firecrawl MCP** | 트렌드/키워드 수집에 사용하는 MCP 서버 |
| **Frame** | 6프레임 쇼츠 구조의 각 화면 단위 (Frame 1~6) |

## G

| 용어 | 정의 |
|------|------|
| **Gemini 3.5 Flash (medium)** | 리서치 엔진으로 사용하는 Google의 LLM. medium 모드로 속도/품질 균형 |
| **골드 레퍼런스** | `reports/20260514_/` — 프레임 디자인과 메타데이터의 절대 기준 폴더 |

## H

| 용어 | 정의 |
|------|------|
| **Hard (P2)** | 두 번째 문제. 높은 난이도. Comment Mode로 정답은 고정 댓글에서만 공개 |
| **Hook** | 영상 첫 프레임(Frame 1)에 표시되는 관심 유도 문구. 10단어 이내, 숫자 포함 (RULE 18) |
| **html2canvas** | HTML 요소를 캔버스(이미지)로 변환하는 JavaScript 라이브러리 |

## I

| 용어 | 정의 |
|------|------|
| **Indigo Glow** | 브랜드 강조 색상 `#6366F1` |
| **IQ Spark** | 프로젝트 브랜드명. IQ/논리 퍼즐 콘텐츠 자동화 플랫폼 |

## L

| 용어 | 정의 |
|------|------|
| **landscape.mp4** | 16:9 가로형 영상 (YouTube Long용) |
| **Link in bio** | SNS 캡션에서 프로필 링크로 유도하는 CTA (RULE 4) |

## M

| 용어 | 정의 |
|------|------|
| **Memory MCP** | 과거 사용 이력(해시태그, 성과 데이터)을 관리하는 MCP 서버 |
| **Mensa** | 상위 2% IQ를 가진 사람들의 국제 단체. 퍼즐 난이도 기준으로 참조 |
| **메타데이터_복사용.txt** | 5채널 카피를 대표님이 복사·붙여넣기 할 수 있도록 정리한 텍스트 파일 |

## P

| 용어 | 정의 |
|------|------|
| **P1** | Puzzle 1. 첫 번째 문제 (Easy). Reveal Mode |
| **P2** | Puzzle 2. 두 번째 문제 (Hard). Comment Mode |
| **portrait.mp4** | 9:16 세로형 영상 (Shorts/Reels/TikTok용) |
| **Phase** | 8-Phase 콘텐츠 제작 프로세스의 각 단계 |

## R

| 용어 | 정의 |
|------|------|
| **Raven** | Raven's Progressive Matrices. 비언어적 IQ 테스트. 퍼즐 유형 참조 |
| **Reveal Mode** | Q1(Easy 문제)에서 다음 프레임(A1)에서 정답을 직접 공개하는 전략 |
| **RULE 0~18** | IQ Spark Studio의 절대 규칙 체계 (RULE.md 참조) |

## S

| 용어 | 정의 |
|------|------|
| **6프레임 구조** | Intro → Q1(Reveal) → A1 → Q2(Comment) → Comment CTA → Final CTA |
| **3중 검증** | 순방향 풀이 + 역방향 검증 + 대안 풀이로 정답 정확성을 보장하는 검증 방법 |

## T

| 용어 | 정의 |
|------|------|
| **투 트랙 LLM** | Gemini(리서치) + Claude(카피라이팅) 이중 LLM 아키텍처 |
| **trend_report.json** | 트렌드 리서치 결과를 저장하는 JSON 파일 |

## U

| 용어 | 정의 |
|------|------|
| **US Audience** | 미국 타겟 시청자. 모든 콘텐츠는 US English로 출력 (RULE 5) |

## W

| 용어 | 정의 |
|------|------|
| **WAIS** | Wechsler Adult Intelligence Scale. 성인 IQ 테스트. 퍼즐 난이도 참조 |
| **Web Audio API** | 브라우저 내장 오디오 API. BGM 60초 재생 + 페이드 제어에 사용 |
